# 🌟 VitalityApp – SDG 3 Health & Well-being
> **A Lightweight Health Ecosystem inspired by Google Fit architecture.**

## Problem Statement
Many individuals in Malaysia struggle with inconsistent health management because they lack a single, centralized tool to track both physical habits and mental reflections. While professional ecosystems like Google Fit exist, they often rely on complex background sensors. 

VitalityApp bridges this gap by providing a manual, high-awareness tracking experience that focuses on the correlation between daily physical activity and mental state.

## Solution
VitalityApp is a mobile application developed using **Jetpack Compose** that helps users maintain a "Quantified Self" by tracking Movement, Nutrition, and Sleep. 

The app provides an easy-to-use platform where users can:
* **Monitor Vitality:** Get a real-time health score.
* **Log Habits:** Use interactive sliders for daily tracking.
* **Reflect:** Record daily moods and personal notes.
* **Set Goals:** Create custom health targets and track progress.
* **Manage Profile:** Track streaks and achievements.

## This Project Demonstrates
* **Material Design UI** using Jetpack Compose components.
* **Multi-screen navigation** using Navigation Compose.
* **Shared state management** using a centralized VitalityViewModel.
* **Data Persistence** without using Room or Firebase (ViewModel + DataStore).
* **State Hoisting** to ensure data survives screen rotation.

## Features Included
* **Dynamic Home Dashboard:** Live vitality score calculation.
* **Weekly Insights:** Data visualization and bar chart analytics.
* **Interactive Journal:** Habit sliders and reflection text fields.
* **Smart Goals System:** Real-time progress bars for custom targets.
* **Gamified Profile:** Achievement grid and day-streak tracking.

## SDG Alignment
### SDG 3 – Good Health and Well-being
This application encourages a holistic approach to health by making habit tracking accessible and intentional. By bridging physical activity with mental reflection, it promotes long-term wellness for the Malaysian community.
